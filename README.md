# Portfolio Risk Architecture

Claude Code marketplace for the `portfolio-risk-architect` skill.

## Install in Claude Code

Add this repository as a marketplace:

```sh
/plugin marketplace add chagkrit/Portfolio-Risk-Architecture
```

Install the skill plugin:

```sh
/plugin install portfolio-risk-architect@portfolio-risk-architecture
```

Use it by asking Claude Code to analyze a portfolio, or invoke the namespaced skill:

```sh
/portfolio-risk-architect:portfolio-risk-architect
```
